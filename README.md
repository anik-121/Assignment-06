1. Name of my Project:
DigiTools Platforms


2. A little Description:
DigiTools is a React-based digital marketplace where users can purchase premium productivity tools. It features a dynamic product catalog, a fully functional cart system, and a clean responsive design built with Tailwind CSS and DaisyUI.


3. Technology that I used:
React, Tailwind CSS, DaisyUI, React Hot Toast,JSON



3. Features about my project:

Dynamic product catalog from JSON
Add to cart / remove from cart
Sticky navbar
Responsive design
